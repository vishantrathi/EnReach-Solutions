## Getting Started

1. Clone the repository:
```sh
git clone https://github.com/vishantrathi/CYBER.git
```

2. Install dependencies:
```sh
npm install
```

3. Start development server:
```sh
npm run dev
```

4. Build for production:
```sh
npm run build
```
